## Note
Actuellement, le site utilise des URLs externes (Unsplash et Wikipedia).
Pour utiliser des images locales, remplacez les URLs dans le fichier HTML par les chemins relatifs vers ce dossier.

Exemple :
```html
<!-- Avant -->
<img src="https://images.unsplash.com/photo-..." alt="">

<!-- Après -->
<img src="assets/images/nom-du-fichier.jpg" alt="">
```
